Section 01

从大语言模型 (LLM)
到自主智能体 (Agent)

本节旨在明确界定“智能体”的概念。传统的 LLM 是一个静态的“大脑”，只能根据输入生成文本。而基于 OpenClaw 构建的 Agent，则为这个大脑赋予了“双手”和“感官”，使其能够在真实环境中自主规划、执行任务并修正错误。

🧠

传统 LLM (纯文本交互)

  • ▹ 被动响应：必须由人类提供明确 prompt。
  • ▹ 孤立环境：无法直接访问互联网或本地文件。
  • ▹ 单步推理：缺乏长期的任务分解和记忆管理。
🦾

OpenClaw Agent (自主执行)

  • ✓ 自主规划： 接收顶层目标，自行拆解步骤。
  • ✓ 环境感知： 能够读取网页、API 状态等外部信息。
  • ✓ 工具使用： 可以调用浏览器、执行代码、操作数据库。
