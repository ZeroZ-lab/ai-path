# 27 这门课最后要带走什么

<div class="mt-8 grid gap-5 md:grid-cols-2 text-left">
  <section class="rounded-3xl border border-slate-200 bg-white p-5 shadow-md">
    <div class="text-sm font-bold text-slate-900">分类能力</div>
    <p class="mt-2 text-sm text-slate-600 leading-6">你不会再把 OpenClaw、Claude Code、Codex 都叫成“差不多的 AI 编程工具”。</p>
  </section>
  <section class="rounded-3xl border border-blue-100 bg-blue-50/80 p-5 shadow-md">
    <div class="text-sm font-bold text-blue-900">架构视角</div>
    <p class="mt-2 text-sm text-slate-600 leading-6">你能用 runtime、loop、skills、memory、risk 这套语言，复述一个 Agent 系统是怎么成立的。</p>
  </section>
  <section class="rounded-3xl border border-purple-100 bg-purple-50/80 p-5 shadow-md">
    <div class="text-sm font-bold text-purple-900">产品判断</div>
    <p class="mt-2 text-sm text-slate-600 leading-6">你知道什么任务值得上高自主权 agent，什么任务更适合 coding shell 或流程框架。</p>
  </section>
  <section class="rounded-3xl border border-emerald-100 bg-emerald-50/80 p-5 shadow-md">
    <div class="text-sm font-bold text-emerald-900">风险意识</div>
    <p class="mt-2 text-sm text-slate-600 leading-6">你会开始主动追问权限、安全、审计和沙盒，而不是只被 demo 的酷炫程度带走。</p>
  </section>
</div>

<div class="mt-8 text-base text-slate-600">
  从今天开始，看 Agent，不要再先问“谁最强”，而要先问“它到底是什么系统”。
</div>

<div class="mt-6 rounded-3xl border border-amber-200 bg-amber-50/80 px-6 py-5 text-left shadow-md">
  <div class="text-sm font-bold text-amber-900">下次看到一个新 Agent，立刻做这 3 步</div>
  <ol class="mt-3 space-y-2 text-sm leading-6 text-amber-800 pl-5">
    <li>先判断它更像聊天助手、coding shell、workflow 平台，还是高自主权 agent。</li>
    <li>再用 5 根轴过一遍：运行位置、能力边界、自主程度、记忆、风险控制。</li>
    <li>最后再问：这个任务值不值得给它这么高的 agency。</li>
  </ol>
</div>

<div class="mt-6 text-center text-[1.05rem] font-bold text-slate-800">
  能把一个新产品放回地图里，这门课才算真正学会。
</div>
