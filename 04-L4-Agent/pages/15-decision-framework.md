## 26 以后再看任何 Agent 产品，都先问这 5 个问题

<div class="mt-6 grid gap-5 md:grid-cols-2">
  <section class="rounded-3xl border border-slate-200 bg-white/80 p-6 shadow-md">
    <div class="text-sm font-bold text-slate-900">1. 它跑在哪里</div>
    <p class="mt-2 text-sm text-slate-600 leading-6">浏览器标签页、IDE、终端、本地守护进程、云端服务，决定了它能碰到什么环境。</p>
  </section>
  <section class="rounded-3xl border border-blue-100 bg-blue-50/80 p-6 shadow-md">
    <div class="text-sm font-bold text-blue-900">2. 它有什么“手”</div>
    <p class="mt-2 text-sm text-slate-600 leading-6">只会生成文本，还是会读文件、控浏览器、跑命令、调 API、连消息平台？</p>
  </section>
  <section class="rounded-3xl border border-purple-100 bg-purple-50/80 p-6 shadow-md">
    <div class="text-sm font-bold text-purple-900">3. 它能自己走多远</div>
    <p class="mt-2 text-sm text-slate-600 leading-6">每步都要你批准，还是能自己规划、观察结果、继续下一步？</p>
  </section>
  <section class="rounded-3xl border border-emerald-100 bg-emerald-50/80 p-6 shadow-md">
    <div class="text-sm font-bold text-emerald-900">4. 它怎么记住事情</div>
    <p class="mt-2 text-sm text-slate-600 leading-6">只有当前窗口上下文，还是有长期记忆、项目状态、可检索历史和控制面？</p>
  </section>
  <section class="rounded-3xl border border-rose-100 bg-rose-50/80 p-6 shadow-md md:col-span-2">
    <div class="text-sm font-bold text-rose-900">5. 它怎么控制风险</div>
    <p class="mt-2 text-sm text-slate-600 leading-6">有没有权限分级、人工确认、沙盒、审计日志？如果一个 agent 很强，却完全不谈这一项，通常说明它更像 demo，而不是系统。</p>
  </section>
</div>

<div class="mt-8 rounded-3xl border border-amber-200 bg-amber-50/80 p-5 shadow-md">
  <div class="text-sm font-bold text-amber-900">把这门课浓缩成一句方法论</div>
  <p class="mt-2 text-sm text-amber-800 leading-6">别先问“这个 Agent 厉不厉害”，而要先问“它属于哪一类、边界在哪里、为什么能做到、出了错怎么收”。只要你能用这 5 个问题现场拆一个新产品，这门课才算真的学会了。</p>
</div>
