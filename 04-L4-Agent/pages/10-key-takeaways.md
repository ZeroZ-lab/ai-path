# 11 这门课最后要带走什么

<div class="mt-8 grid gap-5 md:grid-cols-2 text-left">
  <section class="rounded-3xl border border-slate-200 bg-white p-5 shadow-md">
    <div class="text-sm font-bold text-slate-900">认知升级</div>
    <p class="mt-2 text-sm text-slate-600 leading-6">Agent 不是“更聪明的对话框”，而是“有状态、有工具、有权限的执行系统”。</p>
  </section>
  <section class="rounded-3xl border border-blue-100 bg-blue-50/80 p-5 shadow-md">
    <div class="text-sm font-bold text-blue-900">架构视角</div>
    <p class="mt-2 text-sm text-slate-600 leading-6">一个 Agent 真正难的，不是大模型本身，而是运行时、记忆、工具链和安全边界。</p>
  </section>
  <section class="rounded-3xl border border-purple-100 bg-purple-50/80 p-5 shadow-md">
    <div class="text-sm font-bold text-purple-900">产品判断</div>
    <p class="mt-2 text-sm text-slate-600 leading-6">值得做 Agent 的场景，通常都是多步骤、重复、且能部分自主决策的工作。</p>
  </section>
  <section class="rounded-3xl border border-emerald-100 bg-emerald-50/80 p-5 shadow-md">
    <div class="text-sm font-bold text-emerald-900">现实提醒</div>
    <p class="mt-2 text-sm text-slate-600 leading-6">能力越强，越要讲权限、安全和审计。Agent 不是玩具，而是高风险系统。</p>
  </section>
</div>

<div class="mt-8 text-base text-slate-600">
  下一步，我们就可以从“理解 Agent”走到“设计自己的 Agent 工作流”。
</div>
